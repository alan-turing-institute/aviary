# aviary
Air traffic scenario generation for the [Simurgh](https://github.com/alan-turing-institute/simurgh) project.

## Installation

Install:
```bash
pip install .
```

Developer install:
```bash
pip install -e .
```

## Tests

Run the tests from the project root with:
```bash
python -m pytest
```

or simply:
```bash
pytest
```
